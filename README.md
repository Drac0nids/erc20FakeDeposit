# erc20FakeDeposit
erc20代币假充值测试工具

401认证默认账号密码：security/123456
可在main.py代码中修改

第一次使用，将转账账户私钥修改为自己的钱包私钥，将接收转账地址修改为自己的交易所充值地址。
测试时，钱包中需要有足够的公链原生token用作支付gas费。（BNB、ETH、FTM等）
在修改配置界面，粘贴测试token的合约地址和合约ABI。
